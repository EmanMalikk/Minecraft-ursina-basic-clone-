A tiny Minecraft style block building game made with Python & Ursina.

What is Ursacraft?

Ursacraft is a small 3D sandbox game inspired by Minecraft.
I built it to understand how 3D worlds, blocks, and game interactions work in Python.

You can

Place blocks
Destroy blocks
Walk around a small generated world
Explore how Ursina handles 3D graphics

Tech Stack

Python

Ursina Game Engine
